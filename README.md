## Project Description
This project is an audit of an open source software (Python). It includes study of its origin, license, ecosystem, and comparison with proprietary software. It also includes 5 shell scripts demonstrating basic Linux skills.

---

## Scripts Description

### Script 1: System Identity Report
Displays system information such as user, date, and kernel version.

### Script 2: FOSS Package Inspector
Checks and displays information about Python and prints its purpose.

### Script 3: Disk and Permission Auditor
Checks system directories and displays permissions.

### Script 4: Log File Analyzer
Counts occurrences of the word "error" in a file.

### Script 5: Manifest Generator
Takes user input and generates a simple open source philosophy statement.
